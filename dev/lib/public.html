RandomisationModels.BlockUrn — Type
struct BlockUrn <: MultiArmRandomisationModel

Implements the Block Urn Design (BUD) of Zhao and Wang (2011).

The BUD has conditional allocation probability

\[P_{im} = \frac{w_m\lambda + w_mk_{i-1}-n_{i-1,m}}{W\lambda + Wk_{i-1} - (i - 1)}, \quad k_{i-1} = \min_{m=1,...,M} \left\lfloor \frac{n_{i-1,k}}{w_k} \right\rfloor.\]

RandomisationModels.MassWeightedUrn — Type
struct MassWeightedUrn{Tv<:(AbstractVector), T<:Real} <: MultiArmRandomisationModel

Implements the Mass Weighted Urn Design (MWUD) of Zhao (2015).

The MWU has conditional allocation probability

\[P_{im} = \frac{\max{\alpha w_m - n_{i-1,m} + (i - 1)w_m, 0}}{\sum \max{\alpha w_m - n_{i-1,m} + (i - 1)w_m, 0}}\]
